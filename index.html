Welcome to ImportBot

Seamlessly manage and import your GitHub repositories

Features

  • Import repositories from ZIP files, URLs, or Git
  • Search and fork GitHub repositories
  • Generate custom badges for your projects
  • View trending repositories and Git insights
  • Collaborate on code in real-time
  • Set up preview deployments for your projects

To install ImportBot, you can use our web installer or run our installation script:

curl -o- https://raw.githubusercontent.com/sheikh-developer/ImportBot/main/install.sh | bash
